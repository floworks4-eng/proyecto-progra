# Leonardo Portfolio

Portafolio personal tipo CV creado para una evaluación práctica con Vibe Coding e inteligencia artificial. El sitio presenta el perfil de Leonardo Rojas Rojas, estudiante de Negocios y Ciencia de Datos, con secciones de presentación, habilidades, proyectos académicos, intereses y contacto.

## Tecnologías usadas

- Vite
- React
- JavaScript
- HTML semántico
- CSS personalizado
- Framer Motion
- Three.js
- GSAP
- Netlify

## Uso de IA y Vibe Coding

El proyecto fue desarrollado con apoyo de Codex usando Vibe Coding: se definió primero la idea general, luego se generó la estructura del proyecto, se centralizó la información editable y se iteró sobre diseño visual, animaciones, responsive design, accesibilidad y preparación para despliegue.

## Instalación

```bash
npm install
```

## Ejecución local

```bash
npm run dev
```

Después abre la URL que muestra Vite, normalmente:

```txt
http://localhost:5173
```

## Build

```bash
npm run build
```

## Publicación en Netlify

1. Sube el proyecto a GitHub.
2. Entra a Netlify y elige "Add new site".
3. Conecta tu repositorio.
4. Usa esta configuración:

```txt
Build command: npm run build
Publish directory: dist
```

También puedes publicar con Netlify CLI:

```bash
npm install -g netlify-cli
netlify login
netlify deploy
netlify deploy --prod
```
